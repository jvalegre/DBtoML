Getting ``pyCONFORT``
=====================

GitHub
------

The `pyCONFORT <https://github.com/jvalegre/pyCONFORT>`_ program is available free of charge on GitHub.

pip
---

The ``pyCONFORT`` program can also be installed *via* pip using

.. code:: bash

  $ pip install aqme
